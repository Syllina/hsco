Example usage:

```
$ hsco todo
/hsco/todo> 
```
```
$ hsco
hsco> todo
hsco.todo> 
```
```
/hsco> todo new
Title: ***
Description (optional): ***
Deadline (optional): (how to input time?)
Tags: (empty => current)
Priority:
    Urgency: ***
    Importance: ***
/hsco> todo cd
/hsco> todo ls
(with <id>)
/hsco> todo ls -R
/hsco> todo cat [id](def = 0)
/hsco> todo done [id]
```
